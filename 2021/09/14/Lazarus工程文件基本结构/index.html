Delphi、Lazarus保留字、关键字详解

这边讲下几个类型

  • 应用程序：就是标准的图形化程序
  • 简单程序：顾名思义在标准程序下缩减了一些内容
  • 程序：同上
  • 控制台应用程序：黑窗口程序
  • 库：静态链接库，常说的dll

生成应用程序界面

  • use 是 引用包 类似于 C语言中的#inlcude 不过有所区别
  • unit 是指单元 单元文件中包含已编译的函数 你可以直接使用
  • interface是 单元文件中的 表示 单元文件提供给调用者的 接口函数之类的
  • type 是类型定义
  • private 是私有类
  • public 是 公共类
